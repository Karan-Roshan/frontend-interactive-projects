# GitHub Profile Explorer
A simple and modern **GitHub Profile Finder** built using **HTML, CSS, and JavaScript**.  
This web app allows users to search for any GitHub username and instantly view their public profile information such as avatar, repositories, followers, and following.

# How It Works
1. User enters a **GitHub username**
2. JavaScript captures the input
3. A request is sent to the **GitHub API**
4. The API returns user data
5. JavaScript updates the UI with the fetched information

Example response fields used:
- `avatar_url`
- `name`
- `login`
- `public_repos`
- `followers`
- `following`

# ColorShift Studio
ColorShift Studio is a modern and interactive color theme switcher built using HTML, CSS, and JavaScript.  
Users can select a color palette, and the page background dynamically transforms with smooth transitions.

It features a clean UI, glassmorphism design, gradient color cards, and animated interactions.

- Beautiful gradient color palettes
- Glassmorphism UI design
- Smooth background transition effect
- Responsive layout
- Beginner-friendly JavaScript logic
- Hover and active animations

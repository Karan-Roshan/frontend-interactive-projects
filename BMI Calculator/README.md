# BMI Calculator Web App
A modern, responsive, and stylish **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**.  
This project calculates BMI based on user input (height in cm and weight in kg) and provides instant health category feedback.


## Features
- Clean and modern UI design
- Glassmorphism-inspired card layout
- Responsive design (works on desktop & mobile)
- Real-time BMI calculation
- Health category feedback:
  - Underweight
  - Normal weight
  - Overweight
  - Obese
- Input validation with alerts
- Reset functionality
- Floating Help Guide popup
- Height conversion reference (inches → cm)
- Smooth hover animations and transitions


## Technologies Used
- **HTML5** – Structure of the application  
- **CSS3** – Styling, gradients, animations, layout  
- **JavaScript (Vanilla JS)** – Logic and interactivity

## UI Highlights
- Soft radial gradient background
- Glass-like container with blur effect
- Animated hover effects on buttons
- Smooth popup transition for help guide
- Clear and readable typography (Nunito Font)


## Reset Functionality
The reset button:
- Clears height and weight inputs
- Resets BMI display to `00.00`
- Restores default instruction message

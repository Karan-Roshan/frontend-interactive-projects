# My GitHub ProfileCard
A modern and responsive **GitHub Profile Card Web Application** built using **HTML, CSS, and Vanilla JavaScript**.

This project fetches real-time user data from the GitHub REST API and dynamically displays profile information in a stylish card layout.


## Project Description
**My GitHub ProfileCard** is a simple yet powerful frontend project that demonstrates:
- API integration
- AJAX (XMLHttpRequest)
- JSON data handling
- DOM manipulation
- Clean UI design

It retrieves and displays GitHub user information dynamically.


## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- GitHub REST API


## Features
- Displays Profile Image
- Shows Full Name
- Displays Public Repository Count
- Shows Followers Count
- Clean Dark GitHub-Inspired UI
- Smooth Hover Effects
- Real-Time API Data Fetching


## How It Works
1. JavaScript sends a GET request to:
   https://api.github.com/users/USERNAME
2. GitHub returns user data in JSON format.
3. The application extracts:
   - name
   - public_repos
   - followers
   - avatar_url
4. The DOM updates dynamically with fetched data.
